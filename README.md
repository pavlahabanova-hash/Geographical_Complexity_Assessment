# Geographical_Complexity_Assessment
Design and implement a method to assess the complexity and local uniqueness of small geographic areas

This technical report describes the development, implementation, and experimental evaluation of a comprehensive autonomous visual localization system for unmanned aerial vehicles (UAVs) in GPS-denied or spoofed environments. The proposed system utilizes a semantic map-matching methodology. Input orthophoto maps of the Brno target area are transformed via a deep convolutional neural network based on the U-Net architecture (with a ResNet-18 backbone) into semantic masks representing key topological features: buildings, road networks, and water bodies. These real-time predicted masks are sequentially correlated with a georeferenced database of ground-truth masks derived from OpenStreetMap (OSM) vector data. The report introduces the concept of a "SLAM-Safe Map" based on the network's pixel-wise prediction uncertainty to identify high-risk homogeneous areas (such as forests and open water). The final localization simulation was evaluated under two distinct scenarios: isolated tile matching (success rate of 66,12 %) and spatial context matching using a 3 x 3 tile matrix (success rate of 75,65 %). The experimental results demonstrate that incorporating spatial context drastically reduces localization ambiguity in transitional landscapes, while achieving near 100 % reliability over highly urbanized environments.
